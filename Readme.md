## Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- MongoDB (Local or Atlas)

### Installation

1. **Run API Server:**

   ```terminal
   cd server
   npm install
   npm run dev

   ```

2. **Run React App::**

```terminal
 cd admin
 npm install
 npm run dev
```
